🎫 Ticket Manager API

A pure Node.js ticket management system - no frameworks, just native HTTP

Node.js 22 Pure JavaScript No Framework Vercel Serverless

📋 Features

  • Create, update, and delete tickets
  • View ticket statistics
  • Filter and list tickets
  • RESTful API design
  • File-based JSON persistence

🔌 API Endpoints

GET /api/tickets
List all tickets
POST /api/tickets
Create a new ticket
GET /api/tickets/stats
Get ticket statistics
PUT /api/tickets/:id
Update an existing ticket
PATCH /api/tickets/:id/close
Close a ticket
DELETE /api/tickets/:id
Delete a ticket

💡 Tech Stack

  • Runtime: Node.js 22 (native http module)
  • Testing: Vitest
  • Linting: Biome
  • Deployment: Vercel Serverless Functions
